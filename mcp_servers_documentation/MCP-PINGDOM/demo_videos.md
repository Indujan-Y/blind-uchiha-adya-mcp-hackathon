## 🔐 Credential JSON Payload
Example payload format for sending credentials to the MCP Server which will be used in Client API payload:
```json
{
  "PINGDOM": {
    "api_key": "your-pingdom-api-key",
    "account_email": "your-pingdom-account-email"
  }
}
```
