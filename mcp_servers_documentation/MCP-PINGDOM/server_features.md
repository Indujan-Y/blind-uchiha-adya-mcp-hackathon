# Pingdom MCP Server Overview

## What is the Pingdom MCP Server?
The Pingdom MCP Server is a connector within the Vanij Platform that enables seamless interaction with Pingdom for monitoring, checks, and alerting using the Pingdom API.

---

## Key Features
- ✅ Create, update, and delete Pingdom checks
- ✅ Retrieve check status, probe information, and results
- ✅ Manage contacts and maintenance windows
- ✅ Use authenticated REST calls for secure interaction

---

## Capabilities
| Capability           | Description                                       |
|----------------------|---------------------------------------------------|
| Check Management     | Create, update, delete, and list checks           |
| Probe Info           | Retrieve probe server details                     |
| Results Retrieval    | Get uptime, outage, and performance summaries     |
| Contact Management   | Manage alert contacts                             |
| Maintenance Windows  | Schedule and manage maintenance periods           |

---

## Supported Pingdom API Versions
- Pingdom API v3.1+
- Requires valid Pingdom account and API token

---

## Security Notes
- Authenticated via **Pingdom API token**
- All communications must be secured over HTTPS

---

## Integration Use Cases
- Automated uptime monitoring
- Incident alerting and reporting
- Maintenance scheduling and reporting
